0) 변경되기 이전 (temp = arr[0] => 1 )

2) arr[1] = arr[0] => 2 (이미 위에서 변경을 시켰으므로 이런 식으로는 되지가 않는다)

-> 따라서 이런식으로 해줘야 한다.
